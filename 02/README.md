

### 命令使用
1. npm run clean 删除dist目录
2. npm run watch 开发时候使用
3. npm run build 测试打包的时候使用
4. npm run publish 发布的时候使用（不使用cdn）
5. npm run cdnpublish 发布的时候使用，会先打包并将静态资源post到cdn，
